If vs. Although

IF (Conditions)

Use If when something must happen first.

If you study, you will pass.

如果：表示条件 (If you study -> You pass)

ALTHOUGH (Surprise!)

Use Although when the result is surprising or opposite.

Although it rained, we played outside.

虽然/尽管：表示转折或意外 (Rain -> Usually inside -> But we played outside!)

Comma Rule

Start with If/Although = Use a comma.

If/Although in the middle = NO comma.
